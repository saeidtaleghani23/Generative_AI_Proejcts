## ğŸ­ Loss Functions Used  

CycleGAN optimizes a combination of **four loss functions**:  

| **Loss Type**                           | **Description** |
|-----------------------------------------|---------------|
| **Adversarial Loss** (L2-MSE)           | Ensures the generated images look real. |
| **Identity Loss** (L1-MAE)              | Helps preserve colors and style. |
| **Cycle Consistency Loss (Forward)** (L1-MAE)  | Ensures **Domain1 â†’ Domain2 â†’ Domain1** produces the original image. |
| **Cycle Consistency Loss (Backward)** (L1-MAE) | Ensures **Domain2 â†’ Domain1 â†’ Domain2** produces the original image. |
